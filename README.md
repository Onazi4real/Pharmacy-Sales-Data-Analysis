# 💊 Pharmacy Sales Data Analysis Using Python

## 📌 Project Overview

This project analyzes pharmacy sales data using Python to generate business insights that support inventory management, sales monitoring, and decision-making.

The analysis explores revenue performance, product demand, supplier contribution, stock levels, expiry monitoring, and daily sales trends.

---

## 🎯 Project Objectives

- Analyze total revenue generated
- Identify top-performing products
- Analyze product categories
- Evaluate supplier performance
- Monitor inventory levels
- Detect products requiring restocking
- Identify expired and expiring products
- Analyze daily sales trends
- Identify fast-moving products
- Visualize business insights using charts
